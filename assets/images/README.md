# Gravity Tables Plugin Icons

This directory contains the official icons and graphics for the Gravity Tables plugin.

## Files Created:

### 🎯 **plugin-icon.svg** (256x256)
- **Usage**: Main plugin icon for WordPress plugin directory
- **Format**: Square SVG, scalable
- **Colors**: WordPress blue (#2271b1) with white grid, orange Gravity Forms indicators
- **Design**: Clean grid/table representation matching the `dashicons-grid-view` theme

### 🎨 **gravity-tables-icon.svg** (128x128)  
- **Usage**: General branding, social media, documentation
- **Format**: Circular design with plugin name
- **Colors**: Same color scheme as main icon
- **Design**: More detailed with text label

### 🖼️ **banner-772x250.svg**
- **Usage**: WordPress.org plugin directory banner
- **Format**: Horizontal banner (772x250px standard)
- **Design**: Professional gradient background with plugin info
- **Content**: Plugin name, tagline, key features, author credit

## Design Philosophy:

### **Color Scheme:**
- **Primary**: WordPress Blue (#2271b1)
- **Secondary**: Darker Blue (#135e96) 
- **Accent**: Gravity Forms Orange (#ff6b35)
- **Success**: Green indicators (#4ade80)
- **Base**: Clean white (#ffffff)

### **Visual Elements:**
- **Grid Pattern**: Represents table/data management
- **Orange Dots**: Connection to Gravity Forms
- **Green Accents**: Editing/interactive capabilities
- **Clean Lines**: Professional, modern aesthetic

### **Consistency:**
- Matches WordPress dashicon `grid-view` theme
- Maintains brand consistency across all formats
- Scalable vector graphics for all screen densities

## Usage in Plugin:

The main WordPress menu already uses `dashicons-grid-view`. These additional icons can be used for:
- WordPress.org plugin submission
- Documentation and marketing materials  
- Freemius integration branding
- Website favicon and social media

## Converting to PNG:

To create PNG versions for specific requirements:

```bash
# Using Inkscape (if installed)
inkscape plugin-icon.svg --export-png=plugin-icon-256x256.png --export-width=256 --export-height=256

# Using online converters
# Upload SVG files to: convertio.co, cloudconvert.com, or similar
```

## WordPress.org Requirements:

For WordPress.org submission, you may need PNG versions:
- **Plugin Icon**: 256x256px PNG  
- **Banner**: 772x250px PNG
- **Banner High-DPI**: 1544x500px PNG (2x resolution)

These SVG files provide the perfect source for creating any required formats.
